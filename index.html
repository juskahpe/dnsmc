Windows Üzerinde DNS Değiştirme

Windows bilgisayarınızda DNS ayarlarını değiştirmek için aşağıdaki adımları izleyebilirsiniz:

  1. Başlat menüsünden Denetim Masasını açın.
  2. Ağ ve Paylaşım Merkezine gidin.
  3. Sol tarafta bulunan Bağdaştırıcı ayarlarını değiştir seçeneğine tıklayın.
  4. DNS ayarlarını değiştirmek istediğiniz bağlantıya sağ tıklayıp Özellikleri seçin.
  5. İnternet Protokolü Sürüm 4 (TCP/IPv4)ü seçip Özellikler butonuna tıklayın.
  6. Açılan pencerede Aşağıdaki DNS sunucu adreslerini kullanı seçin.
  7. Tercih edilen ve alternatif DNS adreslerinizi girin. Önerilen DNS adresleri:
    • Tercih edilen DNS sunucusu: 1.1.1.1 (Cloudflare DNS)
    • Alternatif DNS sunucusu: 8.8.8.8 (Google DNS)

  8. Son olarak Tamam butonuna tıklayın ve değişikliklerinizi kaydedin.
Not: DNS değişikliğinizin etkili olması için internet bağlantınızı yeniden başlatmanız gerekebilir.

Mac Üzerinde DNS Değiştirme

Mac bilgisayarınızda DNS ayarlarını değiştirmek için aşağıdaki adımları izleyebilirsiniz:

  1. Sol üst köşedeki Apple menüsüne tıklayın ve Sistem Ayarlarını seçin.
  2. Network simgesine tıklayın.
  3. Listedeki bağlantı listesinde, DNS ayarlarını değiştirmek istediğiniz ağı seçin.
  4. Sağ alttaki Gelişmiş butonuna tıklayın.
  5. Üstteki sekmelerden DNS sekmesine gidin.
  6. DNS sunucu adreslerini değiştirmek için + Butonuna tıklayın ve tercih ettiğiniz DNS adreslerini girin:
    • Tercih edilen DNS sunucusu: 1.1.1.1 (Cloudflare DNS)
    • Alternatif DNS sunucusu: 8.8.8.8 (Google DNS)
  7. Son olarak Tamam butonuna tıklayın değişikliklerinizi kaydedin.
Not: DNS değişikliğinizin etkili olması için internet bağlantınızı yeniden başlatmanız gerekebilir.
